Steps to follow
1. Download this file dyzaro.exe
2. Download configuration file {{apiKey}}.conf
3. Move these files to desired directory [dyzaroDirectory] (Ex:c:\Dyzaro\)
4. Goto command promt and run
		[dyzaroDirectory]\dyzaro.exe --config [dyzaroDirectory]\{{apiKey}}.conf 
Example: in command promot
		c:\> c:\Dyzaro\dyzaro.exe --config c:\Dyzaro\{{apiKey}}.conf
