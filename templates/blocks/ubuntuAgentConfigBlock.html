1.Run the following line in command prompt:
	DY_KEY={{apiKey}} bash -c "$(curl -L https://github.com/Manikandan25/dyzaro/raw/master/installagent.sh)"
2.Choose the plugin you want to monitor, copy the contents provided with respect to those plugins and paste it to {{apiKey}}.conf
3.Run the following line in command line:
	dyzaro --config {{apiKey}}.conf
